# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

### Подготовка к запуску
- Скачайте код
- Создайте файл `wine.txt` и запишите в него продукты
- Файл `wine.txt` должен соответствовать структуре:
```text
# Белые вина


Название: Белая леди
Сорт: Дамский пальчик
Цена: 399
Картинка: images/belaya_ledi.png 
Выгодное предложение

Название: Ркацители
Сорт: Ркацители
Цена: 499
Картинка: images/rkaciteli.png

...


# Красные вина


...
```
- Если на товар есть скидка, то укажите это в описании, написав `Выгодное предложение` 
- Положите его в корень проекта 
- Картинки положить в папку `images/`, расположенную в корне проекта
- Создайте виртуальное окружение и активируйте его
```bash
virtualenv venv
source venv/bin/activate
```
- Установите зависимости 
```bash
pip install requirements.txt
```

### Запуск

- Запустите сайт командой 
```bash
python3 main.py
```
- Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).
